AN OVERVIEW OF NETWORKING

A Network consists of two or more computers usually connected together by cable, all running software that allows them to communicate with each other.

Local Area Network (LAN) is a network of computers in close proximity to one another, usually in the same building or complex.

Wide Area Network (WAN) is a network of computers spread over long distances, which uses long-range telecommunications links.

Minimum Hardware Components of a LAN are:

Two computers

One that acts as a Server to share its resources.

One that acts as a Workstation or Client that will access the shared resources.

A Network Interface Card (NIC) for each computer.

Cabling to connect the computers together.

Minimum Software Components of a LAN are:

A Network Operating System or a Network-Aware Operating System. Appropriate Protocol Drivers.

Network-Aware Applications.

Network-Aware Applications have the ability to accept or send data across the network.

Recognised Networking Models are:

International Standards Organisation (ISO)

Institute Electrical Electronic Engineers (IEEE)

Networking Models Layers are:

7. Application (User Interface)

6. Presentation (Translation)

5. Session (Syncs & Session)

4. Transport (Packets, Flow Control & Error Handling)

3. Network (Addressing, Routing)

2. Data Link (Data Frames to Bits)

1. Physical (Hardware, Raw Bit Data)

Interface is the interaction between adjacent layers.

Protocol is communication between logical layers on 2 computers.

Application Layer is the window for application processes to access the network services.

Presentation Layer translates data and converts the format used to exchange data among networked computers. (This is sometimes known as the Network Translator). It manages Network Security issues by providing data encryption. It provides rules for data transfer and provides compression to reduce the number of bits being transmitted. It has a utility known as the Redirector that redirects I/O operations to resources on a server.

Session establishes, manages and terminates network sessions between different computers. Performs name recognition and security functions. Provides synchronisation between user tasks by placing checkpoints in the data stream. Implements dialogue control to regulate which side transmits.

Transport ensures delivery of error free messages splits up large and joins small messages into manageable packets in order to transmit them efficiently over the network.

Network responsible for addressing messages and translating logical names & addresses into physical addresses. Also determines the route to the destination computer. Manages traffic problems on the network such as switching, routing, and controlling the congestion of data.

Data-Link turns packets into raw data and vice versa, handles data frames between the network and physical layers, responsible for error-free transfer of data frames via the physical layer. It defines the method to transmit and receive data on the network.

Data-Link broken down into 2 sub Layers:

Logical Link Control (LLC) Upper Layer manages data-link communications between devices.

Media Access Control (MAC) Lower Layer communicates directly with the NIC and is responsible for delivering error-free data between two computers.

Physical transmits raw data over the network via cables (electrical/optical) or possibly radio waves. Defines cables, cards and other physical aspects such as NIC attachments to hardware. Defines techniques (encoding and synchronisation) to transfer raw data through the cable. Adds information to the front and end of the data frames, these are known as frame preamble and frame postamble.

IEEE 802 Documents comprise of:

802.2 defined the LLC

802.3 defined the standard for bus networks that uses Carrier Sense Multiple Access with Collision Detect (CSMA/CD).

802.4 defined the standard for token bus networks.

802.5 defined the standard for token ring networks.

 

PHYSICAL CHARACTERISTICS OF A LAN

LAN Cabling Media is able to transfer data between 10 and 100 Mbps depending on the cable types. It comes in 3 types:

Twisted Pair connectors

Coaxial

Fibre-Optic

Twisted Pair consists of 2 insulated strands of copper wire twisted together.

Unshielded twisted pair is used by telephone systems and if it is of a sufficient grade it can be used on networks.

Shielded twisted pair is less susceptible to interference and carries signals for 100 meters.

Coaxial consists of a conductive centre wire, with a wire mesh surround.

Thinnet Coaxial or thin-wire Ethernet is RG-58 cabling about ¼in thick
